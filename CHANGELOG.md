# Changelog

All phase releases are documented here. Individual verse revisions are
tracked in git history on the per-verse YAML files.

## Unreleased

- Repository scaffold
- DOCTRINE.md, METHODOLOGY.md, REVIEWERS.md first drafts

## Release plan

- `v0.1-preview` — Philippians + Philemon (pilot, review pipeline validation)
- `v0.2-pauline` — Romans through Philemon complete
- `v0.3-gospels` — Matthew through Acts
- `v0.4-nt-complete` — General epistles + Revelation (NT complete)
- `v0.5-torah` — Genesis through Deuteronomy
- `v0.6-former-prophets` — Joshua through 2 Kings
- `v0.7-writings` — Psalms, Proverbs, Job, Ruth, Song, Ecclesiastes,
  Lamentations, Esther, Daniel, Ezra-Nehemiah, Chronicles
- `v0.8-latter-prophets` — Isaiah, Jeremiah, Ezekiel, Twelve
- `v1.0-complete` — Full Bible, all phases re-reviewed after external
  scholarly engagement
